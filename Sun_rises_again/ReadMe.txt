==========
The Sun rises again
==========

This map works only with C&C3 Kane's Wrath

Installing:
Extract file Sun rises again to
C:\Documents and Settings\Username\Application Data\Command & Conquer 3 Kane's Wrath\Maps

Uninstalling:
Delete file Sun rises again
