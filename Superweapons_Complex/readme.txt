Map Name: Superweapons Complex
Author Name: Predatore
Max. Players: 4
Game: C&C 3: Kane's Wrath
Type of Map: Skirmish
Description: There is an Ion Cannon and a Nuclear Silo to be captured in this map. Each player has 3 tiberium spikes near the base, and there are 4 places to expand. There is also a subway station in the center which provides strategical transport for your infantery. This map can be played on 2 vs 2 or free for all.
There is also a Juggernout and an Avatar husk to be captured, but you should go fast for them because they will disapper after some time.
