Map Name: Race circuit
Author Name: Predatore
Max. Players: 2
Game: C&C 3: Kane's Wrath
Type of Map: Skirmish
Description: This map features a race circuit with fifteen cars running. There are two places to expand at the center of the circuit, which can be accessed by four bridges. There is also a back path to the main base, that can be the key to an easy victory if the enemy don't defend it.
