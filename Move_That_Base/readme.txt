Map Name: Move That Base
Author Name: Predatore
Max. Players: 8
Game: C&C 3: Kane's Wrath
Type of Map: Skirmish
Description: This map is on a frozen enviroment.All the players start at the center of the map, and has two engineers. If any player want to stay there, their MCV can be easily captured, so there are eight bridges to escape and build a base. There are 16 tiberium spikes, and 8 civilian refineries to capture. This map can't be played with computer players because they don't move the base.
