Map Name: Asteroid
Author Name: Predatore
Max. Players: 8
Game: C&C 3: Kane's Wrath
Type of Map: Skirmish
Description: This map is ambiented on an asteroid night surface. It is intend for slow computers to run an 8 player map fast. It is very simple with great gameplay. The blue tiberium regenerates fast, and there are many tiberium spikes to capture. This map is to be played on 4 vs 4 or 3 vs 3 leaving the corner start locations empty.
