Map Name: Black Volcano
Author Name: Predatore
Max. Players: 4
Game: C&C 3: Kane's Wrath
Type of Map: Skirmish
Description: This is a yellow zone map. There are many volcanoes in eruption. This map has many blue tiberium field, some tiberium spikes, and two EMP control centers. There are two reinforcement bays, wich spawns a veteran stealth walker each one (avatar for NOD, juggernout for GDI, and tripod for SCRIN). This unit will respawn at the reinforcement bay each time it be destroyed. If any player captures the reinforcement bay, and the walker is still alive, then it won't spawn another walker until the first one be destroyed.
